## ✨ 核心功能

| 功能 | 说明 | 赛题对应 |
|------|------|----------|
| 🔬 岗位能力图谱 | AI生成结构化能力图谱（核心/专业/工具/软技能） | 方向2：挖掘岗位能力生成图谱 |
| 📖 个性化学习路径 | 学情诊断 + 自适应路径推荐 + 成长轨迹可视化 | 方向4：个性化自适应学习 |
| 🔭 新岗位发现 | 基于技术趋势发现并定义新兴岗位 | 创新扩展 |
| 💬 智能问答 | 多轮对话，2-3轮追问支持 | 交互友好性要求 |

## 🚀 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python run.py

# 访问
# http://localhost:8000
```

## 📁 项目结构

```
zhihui-agent/
├── app.py                  # FastAPI主入口
├── run.py                  # 启动脚本
├── requirements.txt        # Python依赖
├── render.yaml             # Render部署配置
├── apis/
│   └── routes/
│       ├── skill_map.py    # 能力图谱+新岗发现API
│       ├── learn_path.py   # 学习路径+诊断API
│       └── chat.py         # 智能问答API
├── core/
│   └── agents/
│       ├── skill_map_agent.py  # 图谱Agent（AI+fallback）
│       └── learn_path_agent.py # 路径Agent（AI+fallback）
├── data/

├── static/
│   ├── index.html          # 首页SPA
│   ├── css/style.css       # 暗色主题CSS
│   └── js/app.js           # 交互逻辑
└── docs/                   # 参赛文档
```
