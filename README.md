# 职慧Agent

挑战杯揭榜挂帅 XA-202603 参赛项目 — 职业教育教学实训与岗位技能智能体

**专业群**：软件技术 | **岗位**：前端开发工程师

## 核心功能

1. **岗位能力图谱** — 输入岗位名称，生成结构化能力图谱（核心能力/专业技能/工具技能/软技能）
2. **个性化学习路径** — 学情诊断 + 自适应学习路径推荐 + 成长轨迹可视化
3. **新岗位发现** — 基于技术趋势发现新兴岗位
4. **智能问答** — 多轮对话，2-3轮追问支持

## 技术栈

- 后端：FastAPI + 智谱GLM-4-flash
- 前端：原生HTML/CSS/JS，暗色主题SPA
- 知识库：50条标注来源的专业领域知识点

## 本地运行

```bash
pip install -r requirements.txt
python run.py
```

访问 http://localhost:8000
