本团队郑重声明：

一、**原创性声明**：本作品为参赛团队原创开发，核心代码、知识库、交互设计均为团队独立完成，未抄袭、剽窃他人成果。

二、**AI技术使用合规**：本作品使用智谱AI（北京智谱华章科技有限公司）提供的GLM-4-flash大语言模型API服务，使用行为符合智谱AI服务条款。AI生成内容仅作为辅助参考，所有输出均经过专业知识库校验和fallback机制保障。

三、**知识库来源合规**：本作品嵌入的专业领域知识共50条以上，来源均为公开出版物、国际标准、官方技术文档、国家职业技能标准等，每条知识均标注简要来源（包括文档名称、URL或标准编号）。具体来源包括：

- MDN Web Docs（Mozilla开发者文档）
- 各技术框架官方文档（Vue.js、React、Vite、Webpack等）
- 《Web前端开发职业技能等级标准》
- RFC国际标准（HTTP/9110、WebSocket/6455等）
- OWASP安全测试指南
- Google工程实践文档

四、**隐私保护**：本作品不收集、存储任何个人隐私数据。用户输入的技能信息仅用于当次诊断，不持久化到服务器，不用于任何商业用途。

五、**教育伦理**：本作品输出内容符合教育伦理要求，不包含歧视性、偏见性内容。AI生成内容标注来源标识（warning字段），确保用户知晓内容生成方式。

六、**安全合规**：本作品已实施基本安全措施，包括：输入长度限制、输出熔断机制（500KB上限）、XSS防护、CORS策略等。

七、**责任声明**：本作品仅供职业教育参考使用，不构成任何就业承诺或职业建议的最终依据。用户应结合实际情况综合判断。
