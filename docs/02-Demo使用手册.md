# 职慧Agent 使用手册

## 一、产品概述

职慧Agent是一款面向职业教育专业群的AI岗位技能智能体，聚焦"软件技术"专业群，围绕"前端开发工程师"岗位，提供4大核心功能：

1. **岗位能力图谱** — AI生成结构化能力图谱
2. **个性化学习路径** — 学情诊断+自适应路径推荐
3. **新岗位发现** — 基于技术趋势发现新兴岗位
4. **智能问答** — 多轮对话，2-3轮追问

**访问地址**：https://zhihui-agent.onrender.com  
**首次访问可能需要30秒唤醒**（免费云服务特性）

---

## 二、操作步骤

### 功能1：岗位能力图谱

1. 打开网站，默认进入"能力图谱"页
2. 在"专业群"下拉框选择专业（默认"软件技术"）
3. 在"目标岗位"输入框填写岗位名称（如"前端开发工程师"）
4. 点击"🚀 生成能力图谱"按钮
5. 等待10-15秒，系统生成完整能力图谱
6. 查看结果：
   - **统计卡片**：技能总数、核心能力数、知识点数等
   - **雷达图**：4维能力分布可视化
   - **技能树**：按类别浏览（可点击"核心/专业/工具/软技能"过滤）
   - **知识点**：50+条标注来源的专业知识（支持搜索）
   - **成长路径**：从入门到高级的阶段规划
   - **关联岗位**：点击可跳转生成该岗位图谱

### 功能2：个性化学习路径

1. 点击顶部"学习路径"标签
2. 填写"目标岗位"（如"前端开发工程师"）
3. 填写"已掌握技能"（逗号分隔，如"HTML,CSS,JavaScript基础"）
4. 填写"项目经验"（描述做过的项目）
5. 设置"学习周期"（4-24周，默认12周）
6. 点击"🔍 开始诊断"
7. 等待15-20秒，查看：
   - **匹配度评分**（环形图0-100分）
   - **差距分析**（4维度gap评估+进度条）
   - **学习路径**（分阶段任务+知识点+实践项目）
   - **成长轨迹预测**（折线图）

### 功能3：新岗位发现

1. 点击"新岗发现"标签
2. 输入技术趋势关键词（如"AI大模型,智能体,Agent"）
3. 或点击预设趋势标签快速填入
4. 点击"🧪 发现新岗位"
5. 查看AI发现的新兴岗位：名称、职责、技能要求、应用场景

### 功能4：智能问答

1. 点击"智能问答"标签
2. 在底部输入框输入问题
3. 或点击快捷问题按钮（核心技能/Vue vs React/面试指南/薪资参考）
4. AI回答后可继续追问（支持2-3轮）
5. 按Enter或点击"发送"

## 三、技术架构

```
用户浏览器 → FastAPI后端 → 智谱GLM-4-flash AI
                              ↓ (超时/失败)
                           Fallback预置数据
                              ↓
                          JSON知识库(50+条)
```

- 前端：原生HTML5/CSS3/JS SPA，暗色主题
- 后端：FastAPI + 4个Agent模块
- AI：智谱GLM-4-flash + fallback双保险
- 部署：Render云平台（Python 3.12）
- 知识库：50+条标注来源的专业领域知识
