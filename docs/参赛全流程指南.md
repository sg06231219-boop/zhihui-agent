## 一、报名阶段（6月30日前必须完成）

### Step 1: 注册账号
1. 打开 https://www.tiaozhanbei.net
2. 点击"揭榜挂帅"擂台赛报名入口
3. 用手机号注册账号
4. **选择身份**：学生 → 填写学校全称、学院、专业、学号

### Step 2: 填写报名信息
- **题目名称**：面向职业教育高水平专业群建设的教学实训与岗位技能智能体开发（XA-202603）
- **作品名称**：职慧Agent — 基于AI的职业教育岗位技能智能体
- **团队成员**：
  - 团队负责人：你自己（先填自己）
  - 其他成员：后续补充（最多10人）
  - 建议组队：3-5人最佳，1前端+1后端+1设计/文档+1路演
- **指导教师**：联系学校老师（最多3名）
- **参赛主体**：你所在的学校全称

### Step 3: 盖章
1. 系统生成报名表 → 下载打印
2. 拿到学校相关部门盖章（通常是团委或教务处）
3. ⚠️ 盖章可能需要1-2周，越早越好！
4. 扫描盖章后的报名表
5. 上传扫描件到报名系统

- QQ群：1073736343（报名选手务必加）
- 群内获取赛事最新通知和答疑
